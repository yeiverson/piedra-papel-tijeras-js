# piedra-papel-tijeras-js

Este proyecto es una implementación simple del clásico juego "Piedra, Papel o Tijeras" usando JavaScript.

## Descripción

El juego permite al usuario enfrentarse contra la computadora. El usuario puede elegir entre piedra, papel o tijeras y el programa determinará el ganador según las reglas tradicionales:

- Piedra vence a tijeras.
- Tijeras vence a papel.
- Papel vence a piedra.

## Instalación

1. Clona este repositorio:
   ```bash
   git clone https://github.com/yeiverson/piedra-papel-tijeras-js.git
   ```
2. Abre el archivo `index.html` en tu navegador.

## Uso

- Selecciona tu jugada (piedra, papel o tijeras).
- La computadora elegirá aleatoriamente su jugada.
- Mira el resultado en pantalla.
